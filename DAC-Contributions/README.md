# DAC Contributions

This directory contains infrastructure and ecosystem contributions developed during participation in the DAC testnet environment.

Each contribution is preserved as a standalone archive containing the original implementation, documentation, or technical material related to the corresponding contribution work.

## Included Contributions

### `node-runner-guide-dual-node-cgnat`

Documentation contribution focused on DAC node operation using Dual-Node Single Machine architecture (Windows + WSL2) for CGNAT environments.

Topics include:

- dual-node infrastructure
- static peering
- CGNAT operational strategies
- node stability optimization
- monitoring and troubleshooting

### `dac-wallet-intelligence-layer`

Wallet analytics and portfolio intelligence layer for DAC testnet wallets.

Includes:

- native balance intelligence
- NFT portfolio analytics
- wallet activity profiling
- reputation scoring
- sybil-risk estimation
- testnet wallet intelligence
