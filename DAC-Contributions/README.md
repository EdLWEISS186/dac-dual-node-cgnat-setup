# DAC Contributions

This directory contains independent infrastructure and ecosystem contributions developed during participation in the DAC testnet environment.

The projects included here focus on:

- infrastructure tooling
- node operations
- monitoring systems
- wallet intelligence
- testing utilities
- ecosystem support tooling

Each contribution is maintained as a standalone module or research component and may evolve independently based on ongoing DAC testnet development.

## Included Contributions

### `dac-wallet-intelligence-layer`

Wallet analytics and portfolio intelligence layer for DAC testnet wallets.

Includes:

- native balance intelligence
- NFT holdings resolver
- activity analytics
- portfolio intelligence
- reputation scoring
- sybil-risk profiling

### `node-runner-guide-dual-node-cgnat`

Practical node runner guide for Dual-Node + CGNAT environments.

Includes:

- Windows + WSL dual-node setup
- CGNAT workaround notes
- static peering guidance
- monitoring practices
- stability testing observations
- troubleshooting references
