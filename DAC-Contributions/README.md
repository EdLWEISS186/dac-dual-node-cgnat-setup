# DAC Contributions

This directory contains independent infrastructure and ecosystem contributions developed during participation in the DAC testnet environment.

The projects included here focus on:
- infrastructure tooling,
- node operations,
- monitoring systems,
- wallet intelligence,
- testing utilities,
- and ecosystem support tooling.

Each contribution is maintained as a standalone module or research component and may evolve independently based on ongoing DAC testnet development.

## Included Contributions

- `dac-wallet-intelligence-layer`
  - Wallet analytics & portfolio intelligence layer for DAC testnet wallets
  - Includes NFT activity analytics, reputation scoring, and wallet profiling utilities
