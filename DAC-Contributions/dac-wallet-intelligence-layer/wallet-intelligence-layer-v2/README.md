## What This Project Does

Wallet Intelligence Layer v2.0.2 allows users to:

```text
1. Connect a DAC Testnet wallet
2. Check wallet activity and profile signals
3. Preview a dynamic Wallet Status badge
4. Mint one non-transferable SBT badge
5. Update the badge when wallet status changes
6. View the final SBT inside DAC Sender NFT Launchpad
```

The badge is not just a static NFT image.  
It is generated dynamically from wallet-based on-chain signals.

## Key Features

### Dynamic Wallet Intelligence

The web layer reads and displays wallet-level signals such as:

```text
Native DACC balance
DAC Inception Rank badge count
Wallet profile class
Pattern health
Sybil risk
Inception rank
Wallet archetype
Native balance tier
```

### One Evolving Badge per Wallet

Each wallet can mint only one Wallet Status SBT.

```text
1 wallet = 1 badge
```

If the wallet profile changes later, the badge can be updated instead of minting a new one.

### ERC-5192 Locked SBT

The badge is implemented as a locked SBT.

```text
Transferability: Locked
Token Type     : ERC-5192 SBT
```

Transfers and approvals are disabled.

### Dynamic tokenURI

The final contract uses dynamic metadata.

```text
tokenURI(tokenId)
↓
reads wallet owner
↓
reads current / stored badge profile
↓
generates metadata JSON
↓
generates SVG image
↓
returns base64 encoded on-chain metadata
```

### Dynamic SVG Badge Image

The NFT image is generated dynamically per wallet.

The web preview uses:

```text
dynamicSvgOf(address)
```

This ensures the preview follows the wallet being checked.

## DAC Sender NFT Launchpad Compatibility

Wallet Status SBT is registered into the existing DAC Sender NFT Launchpad registry.

After registration, the collection appears in:

```text
DAC Sender NFT Launchpad
↓
Collections
```

After a user mints, it also appears in:

```text
DAC Sender NFT Launchpad
↓
My Collections
```

The Launchpad `mint.html` was patched to support the dynamic artwork model used by Wallet Status SBT.

Instead of relying only on a static registry image, the Launchpad can render:

```text
dynamicSvgOf(connectedWallet)
```

for the Wallet Status SBT collection.

## Important Notes

This project is built for DAC Testnet.

The Wallet Status SBT is designed as a testnet contribution and wallet intelligence experiment.  
It should not be treated as a financial score, credit score, or security guarantee.

The classification is based on visible on-chain signals and heuristic interpretation, including DAC Inception Rank signal and native DACC balance tier.
