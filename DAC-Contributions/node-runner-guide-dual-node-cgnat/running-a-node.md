# Running a Node

Guide for running DAC Quantum Chain nodes.

## Overview

This guide focuses on the **Dual-Node Single Machine** setup (Windows + WSL2), which is effective for maximizing resource utilization and network contribution, particularly for users behind CGNAT.

## Hardware Requirements

**Minimum**
- CPU: 4 cores
- RAM: 8 GB
- Storage: 500 GB NVMe SSD
- Internet: 50 Mbps symmetric

**Recommended**
- CPU: 8+ cores
- RAM: 16–32 GB
- Storage: 1 TB+ NVMe SSD
- Internet: 100 Mbps+ symmetric

## Recommended Setup

**Dual-Node Single Machine (Windows + WSL2)**

**Benefits:**
- Higher resource utilization
- Internal peering capability
- Better redundancy and stability
- Optimized for CGNAT environments
