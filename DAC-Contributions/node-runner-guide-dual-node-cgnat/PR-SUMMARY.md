## Summary

This contribution adds a practical infrastructure guide for running DAC nodes in constrained network environments.

The guide focuses on a Dual-Node setup using:

- Windows host
- WSL/Linux environment
- CGNAT workaround
- static peering
- monitoring practices
- stability testing
- troubleshooting guidance

## Purpose

The purpose of this contribution is to support DAC testnet participants who want to operate stable nodes with limited hardware or restricted network conditions.
